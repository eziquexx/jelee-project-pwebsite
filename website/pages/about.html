기술 skills

HTML5

HTML5를 사용하여 웹 페이지의 구조를 설계할 수 있습니다.

CSS3

CSS3를 활용하여 웹 페이지의 콘텐츠 요소를 적절히 배치하고, 반응형 웹 디자인을 구현할 수 있습니다

jQuery

jQuery를 활용하여 동적인 웹 요소를 구현할 수 있습니다.

예를 들어, 슬라이드 배너와 스크롤 이벤트를 통해 애니메이션 효과를 적용할 수 있습니다.

또한, 구글 검색, W3Schools, jQuery 공식 문서 등을 참고하여 필요한 기능을 추가할 수 있습니다.

JavaScript

기본 문법은 이해하고 있으나, 구현 과정에서 유연함이 부족하고 시간이 걸립니다.

JavaScript 기능을 구현할 때는 MDN, 구글 검색, W3Schools와 같은 자료를 참고하고 있습니다.

원하는 기능 구현에 어려움이 있을 경우에는 ChatGPT를 활용합니다.

React

현재 React의 기본 개념 중 Component의 역할과 useState, useEffect 훅의 기본적인 사용법을 이해하고 있습니다.

또한, React를 활용하여 미니 프로젝트로 Todo List를 개발해보았으며, 이 과정에서 구현 방식에 대한 이해를 높였습니다.

ETC
